Повторное открытие: что видит авторизованный пользователь

Сценарий: пользователь уже сохранил результат («Кешбэк 6.2026») и снова открывает приложение. Четыре варианта первого экрана. Жёлтым подсвечены ключевые точки входа.

Общее для всех: редактирование = открыть сохранённый результат → «Загрузить ещё» → bank-select с уже привязанными скриншотами → processing → обновлённые кешбэки → пересохранить.
